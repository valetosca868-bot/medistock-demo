======================================
    MEDISTOCK PRO - DEMO COMPLETO
======================================

CONTENIDO DEL PAQUETE:
----------------------
1. demo.html                    - Página principal del demo
2. demo-styles.css              - Estilos principales
3. demo-styles-extra.css        - Estilos adicionales
4. demo-script.js               - Funcionalidad principal
5. demo-script-extra.js         - Funcionalidades adicionales
6. demo-script-reports.js       - Sistema de reportes
7. demo-data-generator.js       - Generador de datos de prueba


INSTRUCCIONES DE USO:
---------------------
1. Extraer todos los archivos en una carpeta
2. Abrir el archivo "demo.html" en un navegador web moderno
   (Chrome, Firefox, Edge, Safari)
3. El demo funciona completamente offline, no requiere servidor


CARACTERÍSTICAS DEL DEMO:
-------------------------
✓ Sistema de gestión hospitalaria completo
✓ Dashboard con estadísticas en tiempo real
✓ Gestión de inventario
✓ Gestión de pacientes
✓ Atención al paciente
✓ Gestión de proveedores
✓ Control financiero
✓ Gestión de camas hospitalarias
✓ Sistema de reportes
✓ Cierre mensual e historial
✓ Diseño responsive (funciona en móviles y tablets)


NAVEGACIÓN:
-----------
- Usar el menú lateral para navegar entre secciones
- El botón de toggle colapsa/expande el sidebar
- En móviles, usar el botón de menú hamburguesa


DATOS DE PRUEBA:
----------------
El demo incluye datos de prueba generados automáticamente:
- Pacientes con cédulas de 8 dígitos
- Inventario de medicamentos y suministros
- Transacciones financieras
- Ocupación de camas
- Reportes mensuales


TECNOLOGÍAS UTILIZADAS:
-----------------------
- HTML5
- CSS3 (con variables CSS y gradientes)
- JavaScript Vanilla (sin frameworks)
- SVG para iconos
- LocalStorage para persistencia de datos


COMPATIBILIDAD:
---------------
✓ Chrome 90+
✓ Firefox 88+
✓ Safari 14+
✓ Edge 90+

NOTAS IMPORTANTES:
------------------
