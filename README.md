# MediStock Pro - Demo

Sistema completo de gestión hospitalaria desarrollado con tecnologías web modernas.

## Características

- Dashboard con estadísticas en tiempo real
- Gestión de inventario de medicamentos y suministros
- Gestión de pacientes
- Atención al paciente
- Gestión de proveedores
- Control financiero
- Gestión de camas hospitalarias
- Sistema de reportes
- Cierre mensual e historial
- Diseño responsive (móviles y tablets)

## Tecnologías

- HTML5
- CSS3 (variables CSS y gradientes)
- JavaScript Vanilla (sin frameworks)
- SVG para iconos
- LocalStorage para persistencia de datos

## Demo en Vivo

Visita la demo: [Ver Demo](https://tuusuario.github.io/medistock-demo/)

## Uso Local

1. Clona este repositorio
2. Abre `index.html` en tu navegador
3. Explora todas las funcionalidades
